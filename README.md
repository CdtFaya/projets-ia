# Projet IA - Expert Advisor MetaTrader 5

Ce workspace contient la structure locale du projet et les sources de l'Expert Advisor.

## Structure
- `src/` : sources et futurs modules d'IA / backtesting
- `metatrader/MQL5/Experts/ProjetsIA/` : sources MQL5 prêtes à être copiées dans MetaTrader 5

## Utilisation
1. Copier le contenu de `metatrader/MQL5/Experts/ProjetsIA/` dans le dossier `MQL5/Experts/` de votre installation MetaTrader 5.
2. Compiler l'EA depuis MetaEditor.
3. Déposer l'EA sur un graphique et le tester.

## Versioning
Le dépôt Git local est initialisé dans ce répertoire.
